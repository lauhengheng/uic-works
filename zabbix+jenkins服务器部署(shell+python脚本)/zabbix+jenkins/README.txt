  这个项目是本人做运维工程师工作期间部署的zabbix服务器监控平台和jenkins ci/cd自动化部署工具的搭建文档，具体如下：
  搭建时间：2020.6
  
  zabbix项目（实现可视化监控服务器资源，资源达到阈值后的自动化告警的等功能）：
  1.搭建zabbix监控服务器的搭建流程
  2.操作zabbix监控平台利用shell脚本切割服务器实时资源数据，并配置监控策略实现监控平台可视化监控服务器资源（如cpu，
  内存，文件系统，服务器流入、流出流量，TCP连接状况，微服务工作状态，接口调用情况的监控等）。
  3.编写python脚本实现服务器资源达到阈值后的自动化告警功能（调用钉钉机器人接口实现资源达到触发器的阈值后利用钉钉机器人发消息）
  
  jenkins项目（实现项目自动化打包、部署等一系列操作）：
  1.搭建jenkins服务器
  2.配置jenkins与git服务器的ssh无密连接
  3.利用jenkins的插件完成到git上自动打包的流程
  4.编写shell脚本实现打包后的自动化操作
    （1）从git上自动将应用包下载到生产服务器
    （2）验证包的文件完整性（MD5值）
    （3）标明应用包的版本号
    （4）自动部署项目至服务器
    （5）回滚机制
